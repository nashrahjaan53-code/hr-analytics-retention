# HR Analytics & Employee Retention Analysis

## Overview
Comprehensive HR analytics platform analyzing employee performance, retention patterns, attrition prediction, and workforce metrics to optimize human resources management.

## Features
- **Employee Performance Tracking**: Monitor KPIs and ratings
- **Attrition Prediction**: Identify employees at risk of leaving
- **Retention Analysis**: Understand reasons for turnover
- **Department Analytics**: Compare metrics across teams
- **Salary Equity Analysis**: Identify pay gaps

## Project Structure
```
├── data/              # Employee and HR data
├── src/               # Analysis modules
├── notebooks/         # HR analytics notebooks
├── requirements.txt   # Dependencies
└── main.py           # Main analysis script
```

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Key Metrics
- Employee Retention Rate
- Attrition Rate by Department
- Average Tenure
- Performance Scores
- Salary Distribution
